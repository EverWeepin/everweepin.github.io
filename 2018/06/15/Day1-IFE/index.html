Day 1-IFE

2018-06-15
Front-End, Notes

学习的目标是
希望通过技术做出类似这些美好的东西，
用技术带给用户最优美的体验。

CSS 代码的格式基本是 属性:值 。
position:fixed;
对应的代码翻译成人话，就是它的位置是固定不动的。

浏览器会根据 CSS 代码 ，“画”出对应的样式。

XHTML是 HTML 的近亲 XML 和 HTML 自己的杂交品种。

HTML定义内容，CSS规定样式，JS实现动态体验。
html是名词，css是形容词，
javascript是动词。三个互相配合才是一句句子

看了资料里给的链接，其中有几个能根据给定的可变的参数生成图像
让我十分受触动，只需要修改一些参数就能让结果看起来如此不同，
第一个artsy的网站真的把我惊艳到了，希望结束在这里的学习之后
我也能做出这样能让人眼前一亮的网页。
很干些百度能提供给我们这样的优质学习资源，

未来的66天我一定能坚持下去，
到最后我一定能做出比artsy还要更好看的网站!
